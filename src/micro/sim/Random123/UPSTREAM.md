# Random123 Source Integration

This directory contains Random123 headers copied from the NEURON 9.0.1 wheel:

- Source path during import: NEURON wheel/include installation, `neuron/.data/include/Random123`
- Role in MIND_Sim: header-only dependency required by vendored CoreNEURON random and generated mechanism code.

Random123 files carry their own BSD-style copyright/license notice in the source headers. MIND_Sim keeps those notices intact.
